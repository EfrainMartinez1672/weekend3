# Inventory Management App

## Description
This is a simple **inventory management app** written in Python.  
You can **add, view, search, update, delete products**, calculate inventory statistics, and **save/load CSV files**.

---

## Features
- Add products with **name, price, and quantity**
- Show all products in a **clean format**
- Search product by **name**
- Update **price or quantity** of a product
- Delete product from inventory
- Calculate **total value** of inventory
- Save inventory to **CSV file**
- Load inventory from **CSV file** (overwrite or merge)

---

## Requirements
- Python 3.x

---
